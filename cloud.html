Comparing the Effect of Latitude on Cloudiness

This anlysis looked for a relation between latitude and cloud cover(%). Using a Pandas DataFrame that contained the location and cloud cover data for each city, a scatter plot was plotted using MatPlotLib to demonstrate the relation. There was no relationship found between longitude, or distance from the equator, and cloud cover.
